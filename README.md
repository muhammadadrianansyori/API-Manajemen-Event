# API-Manajemen-Event
Proyek ini merupakan pengembangan RESTful API untuk Sistem Manajemen Event dan Pendaftaran Peserta sebagai tugas UAS mata kuliah Pemrograman Web Service. API ini menyediakan fitur autentikasi JWT, pengelolaan data event, pendaftaran peserta, serta pencatatan log aktivitas menggunakan Laravel dan MySQL.
